# Minsky Financial Instability Hypothesis: Keen Model

## Project Overview
This repository contains a Python implementation of Steve Keen's mathematical formalization of Hyman Minsky's Financial Instability Hypothesis. Originally developed as an academic project, the scripts model the non-linear macroscopic dynamics of private debt, wage shares, and employment rates.

## Model Features & Implementation
* **Dynamical Systems Modeling:** Uses systems of ordinary differential equations (ODEs) inspired by Lotka-Volterra predator-prey dynamics to represent the interaction between economic classes.
* **2D and 3D Phase Space Analysis:** Includes distinct Python scripts to integrate and analyze both the simplified 2-equation system and the complete 3-equation system, tracking the evolution of the economy towards equilibrium or debt-induced crises.
* **Academic Report:** The repository includes the original project report (`Relazione_Keen_Minsky.pdf`) detailing the theoretical framework and the mathematical derivations.

## Tech Stack
* **Language:** Python.
* **Libraries:** SciPy (odeint), NumPy, Matplotlib.
* **Methodology:** Ordinary Differential Equations (ODEs), Phase Space Analysis, Complex System.
